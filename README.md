# Polymor-Farm-Game-Edukatif-OOP
Game edukatif berbasis Python yang menerapkan konsep Object-Oriented Programming (OOP) untuk anak sekolah dasar

Polymor-Farm merupakan game edukatif sederhana berbasis Python yang dirancang sebagai media pembelajaran interaktif untuk anak sekolah dasar. Game ini menggabungkan unsur hiburan dan edukasi (edutainment) dengan penerapan konsep **Object-Oriented Programming (OOP)** secara nyata dan visual.

## ğŸ§  Konsep OOP yang Diterapkan
Game Polymor-Farm menerapkan konsep dasar OOP sebagai berikut:
- **Encapsulation**  
  Pengamanan data atribut hewan (kesehatan, kebahagiaan, energi)
- **Inheritance**  
  Class `Animal` diturunkan menjadi `Chicken`, `Cow`, dan `Sheep`
- **Polymorphism**  
  Method yang sama memiliki perilaku berbeda (produksi telur, susu, wol)
- **Abstraction**  
  Method abstrak pada class induk untuk memastikan implementasi pada class turunan
